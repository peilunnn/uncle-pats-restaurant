# Setup

1. Set up frontend

   From root,

   ```
   cd frontend/online-store
   npm install --legacy-peer-deps
   npm start
   ```

2. Set up backend

   From root,

   ```
   cd backend
   npm install --legacy-peer-deps
   npm start
   ```

3. Set up SQLite DB
   From root,

   ```
   cd backend
   node db.js
   ```

4. Load dummy data into DB
   ```
   node loadDummyData.js
   ```
